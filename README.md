## 충돌방지 시스템 데이터 로그
- 폐사에서는 공식적으로 충돌방지 시스템 데이터 백업 및 조회를 지원하지 않습니다.
- 데이터 저장 및 조회(시각화, 분석) 서비스가 필요하시면, REST API 로 데이터 연동 지원 드리겠습니다.
- 다만, 자사에서 운영상 필요로 인한 파일 로그를 우선 샘플로 전달 드리겠습니다. 페이로드 규격은 첨부된 시트 에서 확인 가능합니다.

### 자동 감지형 충돌방지 시스템
* time: 데이터 수신 시각 (YYYY/MM/DD hh:mm:ss)
* topic: 타워크레인 관리코드
* JL: 작업반경
* WL: 충돌위험 단계 (1~5단계)
  * 0: 20m 이상	안전거리 유지	초록
  * 1: 20m ~ 15m, 충돌경고, 노랑
  * 2: 15m ~ 10m, 충돌주의, 주황
  * 3: 10m ~ 5m, 충돌위험, 빨강
  * 4: 5m 미만, 충돌위험, 진한빨강  
* DT: 인접한 타워 중 최단거리 및 타워크레인의 호기명
* CS: 컴파스 센서 정보, 회전각
* SS: 기울기 센서 정보, 경사각
* TR: 트롤리 위치 (m 단위)
* TRCNT: 트롤리 엔코더 카운터

### 네트워크 무선 풍속계
* curSpeed: 현재 풍속
* avgSpeed: 평균 풍속
* maxSpeed: 최대 풍속
* minSpeed: 최소 풍속
* 네트워크 풍속계 알림 단계
  * 0: 10m/s 이하 작업가능
  * 1: 10m/s ~ 15m/s 작업주의
  * 2: 15m/s 이상 작업중지
